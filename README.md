## 📝 О проекте

**NN Launcher** — это графическая оболочка (веб-интерфейс) для утилиты **ZAPRET**, разработанной [Flowseal](https://github.com/Flowseal/zapret-discord-youtube).

### Что сделал я:
- Создал удобный графический интерфейс (лаунчер)
- Добавил управление стратегиями через кнопки
- Сделал редактор списка сайтов
- Добавил автозапуск и автоподключение
- Реализовал систему обновлений
- Добавил логирование и диагностику

> **Важно:** Весь функционал обхода блокировок обеспечивает ZAPRET — я только сделал удобную оболочку для его запуска и управления.

## 📂 Структура приложения

```
NN Launcher/
├── NN_Launcher.exe          # Основной лаунчер
├── ZAPRET/                  # Папка для стратегий
│   ├── *.bat                # Ваши .bat файлы
│   └── lists/               # Папка со списками сайтов
│       └── list-general.txt # Список сайтов для обхода
└── vpn_config.json          # Файл настроек
```

## ❓ Решение частых проблем

<details>
<summary><b>🔴 ZAPRET не запускается</b></summary>

- Запустите лаунчер от имени администратора
- Временно отключите антивирус
- Убедитесь, что в папке ZAPRET есть .bat файлы
</details>

<details>
<summary><b>🟡 Не обходит блокировки</b></summary>

- Попробуйте другую стратегию из списка
- Обновите список сайтов (кнопка "РЕДАКТИРОВАТЬ САЙТЫ")
- Проверьте логи в нижней панели
</details>

<details>
<summary><b>🟠 Не сохраняются настройки</b></summary>

- Убедитесь, что есть права на запись в папке приложения
- Проверьте, что файл vpn_config.json не заблокирован
</details>

## 🔧 Для продвинутых пользователей

### Добавление своих стратегий

1. Создайте `.bat` файл со своей конфигурацией ZAPRET
2. Поместите его в папку с лаунчером или в папку `ZAPRET`
3. Перезапустите лаунчер — стратегия появится в списке

### Редактирование списка сайтов

- Используйте кнопку **"РЕДАКТИРОВАТЬ САЙТЫ"**
- Каждая строка — отдельный домен
- После сохранения изменения применяются сразу
