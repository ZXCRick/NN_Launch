# Changelog

## [1.0.6] - 2024-11-24

### Улучшено
- ZAPRET починен под новые алгоритмы и автоматизирован для останокаи дальнейших сбоев

### Исправления
- Добавлено больше функций в настройки запрета
- Всё ещё грядёт большое обновление и перепись VPN

## [1.0.5] - 2024-11-24

### Улучшено
- Полная адаптация под новые алгоритмы РКН - обход работает даже с обновлёнными системами DPI-блокировок
- Усиленные методы десинхронизации - увеличены повторения и улучшена стабильность соединения
- Оптимизированные параметры обхода - подобраны оптимальные настройки для текущих реалий блокировок

### Исправления
- Решены проблемы с обрывом соединения
- Устранены конфликты с современными системами блокировок
- Улучшена совместимость с различными провайдерами
- Скоро грядут масштабные обновления - готовлю интеграцию с лаунчером для максимально удобной работы
